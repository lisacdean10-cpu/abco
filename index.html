Setup instructions
One-time setup (5 minutes):
1. Go to console.cloud.google.com
2. Create a new project (name it "ABCo")
3. Enable Google Drive API
4. Go to APIs & Services → Credentials
5. Create OAuth 2.0 Client ID → Web application
6. Add your domain to Authorized JavaScript origins
   (or add http://localhost for local use)
7. Copy the Client ID and paste it above
8. Tap Connect Google Drive and sign in

Your data file will appear in Google Drive as ABCo_StudioData.json.
